📚 Wikipedia Summary App (Python GUI)

This project is a simple GUI-based application that allows users to search for any topic and retrieve a summary from Wikipedia using Python's `wikipedia` module. It's built using the `tkinter` library for the graphical interface.

🖼 Features

- 🔍 Search any topic directly from the GUI
- 📄 Display summarized information from Wikipedia
- 🪶 Clean and user-friendly interface
- 💡 Handles invalid or missing input gracefully

🛠 Technologies Used

- Python 3.x
- `wikipedia` module
- `tkinter` for GUI

📦 Installation

1. **Clone the repository**
      bash
   git clone https://github.com/your-username/wikipedia-gui-summary.git


2. **Navigate to the project folder**
      bash
   cd wikipedia-gui-summary
   

3. **Install required packages**
      bash
   pip install wikipedia


🚀 Usage

Run the app with:
   bash
python wiki_gui.py


Enter a search term and click the "Search" button to view a short summary from Wikipedia.
